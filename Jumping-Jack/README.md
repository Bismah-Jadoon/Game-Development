# 🕹️ Unity 2D Platformer Game

A 2D platformer game built in **Unity** where the player controls an anime character that can **run, jump, turn**, and **die when touching traps**.

⚠️ **Important Note:**  
This project is created as a **practice/learning project**. I followed a YouTube tutorial to build this game and understand Unity mechanics. It is **not an original commercial game**, but part of my learning journey in game development.


## 📹 Tutorial Reference (Learning Source)

This game was made while learning from the following YouTube tutorial:

👉 https://youtu.be/Ii-scMenaOQ?si=3yLZAFld-m6hC2Sq

Credit goes to the original creator for the guidance and teaching.

## 🧠 Game Overview

In this game:

✔️ The character can **run and jump**  
✔️ The character **turns direction** based on input  
✔️ The character **dies when hitting traps**  
✔️ Simple animations are triggered for movement & death


## ▶️ How to Play

| Action | Input |
|--------|-------|
| Move Right | **D** / **Right Arrow** |
| Move Left | **A** / **Left Arrow** |
| Jump | **Space** |

> You can change these controls in Unity’s Input settings.

## 🛠️ Features

### Player
- **Movement:** Run left/right
- **Jumping:** Single jump
- **Animation:** Idle, Run, Jump, Dead
- **Collision:** Detects traps and triggers death

### Environment
- **Platforms**
- **Traps**
- **Level boundaries**
